# Criterios de calidad de datos de agua

## Niveles

- **Alta**: medición directa con evidencia clara.
- **Media**: factura o respaldo razonable.
- **Baja**: estimación u origen poco trazable.

## Señales de alerta

- volumen negativo
- retornos superiores a la captación
- reúso sin justificación
- factores de escasez sin método, versión o fuente
- factor sin correspondencia geográfica

## Resultado

Cuando la calidad de datos es insuficiente, el sistema debe mostrar:

- inventario físico disponible
- huella de escasez no disponible o pendiente

sin mezclar m³ con m³-eq.
