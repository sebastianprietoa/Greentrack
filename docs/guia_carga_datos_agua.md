# Guía de carga de datos de agua

## Datos a registrar

### Captación

- sede
- mes/año
- fuente de agua
- volumen en m³
- área o proceso
- calidad de dato
- evidencia
- observaciones

### Retornos y reúso

- sede
- mes/año
- tipo: retorno o reúso interno
- volumen en m³
- destino
- retorno al mismo sistema hídrico
- tratamiento
- proceso
- evidencia
- observaciones

## Recomendaciones

- No registrar volúmenes negativos.
- No asumir que un factor aplica a una cuenca si el código geográfico no coincide.
- Completar evidencia documental cuando sea posible.
- Separar siempre captación, retorno y reúso.

## Buenas prácticas

- Usar medición directa cuando exista.
- Si se usa estimación, dejarlo explícito.
- Mantener el período mensual consistente para toda la sede.
